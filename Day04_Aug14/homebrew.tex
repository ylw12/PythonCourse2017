1. Type in the Terminal
	* brew install selenium-server-standalone

2. Download ChromeDriver: https://sites.google.com/a/chromium.org/chromedriver/
3. Open up Terminal
4. Run: sudo nano /etc/paths
5. Enter your password
6. Go to the bottom of the file and enter the path you wish to add
7. The PATH should look like: /Users/name/Documents/WebDriver
8. Control-x to quit
9. Y to save
10. Press enter to confirm
11. To double check, quit Terminal and relaunch
12. Run echo $PATH. You should see your newly added path in the stream of other paths already there.
